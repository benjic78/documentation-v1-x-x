# Hardware para Pypilot

## ¿Qué necesita?

* Una unidad de medición inercial \(IMU\) MPU-9255 or MPU-9250. El HAT Moitessier está equipado con la MPU-9250
* Un controlador de motor para accionar un motor de escobillas
* Un motor eléctrico
